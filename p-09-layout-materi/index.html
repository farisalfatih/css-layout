Layout

Dalam konteks CSS (Cascading Style Sheets), "layout" merujuk pada pengaturan tata letak atau struktur halaman web. Layout CSS digunakan untuk mengatur posisi, ukuran, dan hubungan antara elemen-elemen HTML di dalam halaman. Layout mencakup pemosisian elemen, pengaturan kolom, dan penanganan responsif terhadap perubahan ukuran layar atau perangkat.

Ada beberapa teknik yang dapat digunakan untuk mencapai layout dalam CSS, termasuk:

floats:

Menerapkan properti float pada elemen untuk memposisikannya ke sisi kiri atau kanan dari kontainer induk.

.example {
float : left;
}

flexbox:

Sebuah model tata letak baru yang memungkinkan untuk mengatur elemen dalam suatu kontainer baik secara horizontal maupun vertikal.

.example {
display : flex;
justify-content : space-between;
}

grid:

Sebuah sistem tata letak dua dimensi yang memungkinkan untuk membuat layout kompleks dengan mudah.

.example {
display : grid;
grid-template-columns : repeat(3, 1fr);
}

positioning:

Menggunakan properti position untuk mengendalikan posisi elemen.

.example {
position : absolute;
top : 0;
left : 0;
}

Responsive Design:

Menggunakan media queries dan unit ukuran yang responsif untuk membuat tata letak yang dapat beradaptasi dengan perubahan ukuran layar.

@media screen and (max-width: 600px) {
/* Styles for small screens */
}
